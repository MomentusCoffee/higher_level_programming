# Python - More Data Structures

### Description
In this project lesson you will learn:

* What are set and how to use them

* What are the most common methods of set and how to use them

* When to use sets versus lists

* How to iterate into a set

* What are dictionary and how to use them

* When to use dictionaries versus lists or sets

* What is a key in a dictionary

* How to iterate into a dictionary

* What is a lambda function

* What is map, reduce and map functions

### File Content
This repository contains the following files*:

| File | Task |
| :--- | :--- |
| 0-square_matrix_simple.py | A function that computes the square value of all integers of a matrix from 0-main.py |
| 1-search_replace.py | A function that replaces all occurrences of an element by another in a new list from 1-main.py |
| 2-uniq_add.py | A function that adds all unique integers in a list (only once for each integer) from 2-main.py |
| 3-common_elements.py | A function that returns a set of common elements in two sets from 3-main.py |
| 4-only_diff_elements.py | A function that returns a set of all elements present in only one set from 4-main.py |
| 5-number_keys.py | A function that returns the number of keys in a dictionary from 5-main.py |
| 6-print_sorted_dictionary.py | A function that prints a dictionary by ordered keys from 6-main.py |
| 7-update_dictionary.py | A function that replaces or adds key/value in a dictionary from 7-main.py |
| 8-simple_delete.py | A function that deletes a key in a dictionary from 8-main.py |
| 9-multiply_by_2.py | A function that returns a new dictionary with all values multiplied by 2 from 9-main.py |
| 10-best_score.py | A function that returns a key with the biggest integer value from 10-main.py |
| 11-mutiply_list_map.py | A function that returns a list with all values multiplied by a number without using any loops from 11-main.py |
| 12-roman_to_int.py | A function that converts a Roman numeral to an integer from 12-main.py |
| 100-weight_average.py | A function that returns the weighted average of all integers tuple from 100-main.py |

| 102-complex_delete.py | A function that deletes keys with a specific value in a dictionary from 102-main.py |

> All files have been tested and compiled on Ubuntu 14.04 LTS
