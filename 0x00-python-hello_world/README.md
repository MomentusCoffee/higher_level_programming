## Coding Boot Camp Material

# Python - Hello World

### Description
In this project lesson you will learn:

* Who created Python

* Who is Guido van Rossum

* Where does the name ‘Python’ come from

* What is the Zen of Python

* How to use the Python interpreter

* How to print text and variables using `print`

* How to use strings

* What are indexing and slicing in Python

### File Content
This repository contains the following files*:

| File | Task |
| :--- | :--- |
| 0-run | A Shell Script that runs a Python Script saved in an environment variable $SCRIPT |
| 1-run_inline | A Shell Script that runs Python code |
| 2-print.py | Prints a single double quote in the given print statement |
| 3-print_number.py | Prints a message and the integer stored in the variable number |
| 4-print_float.py | Prints a floating number stored in the variable numbery |
| 5-print_string.py | Prints the string stored in variable str 3 times followed by the first 6 characters |
| 6-concat.py | Prints a concatenated string |
| 7-edges.py | Prints out slices of a string stored in the variable word |
| 8-concat_edges.py | Prints the new string stored in the variable str after being overwritten |
| 9-easter_egg.py | Easter Egg: Python Script that prints The Zen of Python by Tim Peters |
| 10-write.py | A Python script that prints exactly “History Doesn't Repeat Itself, but It Often Rhymes” – Mark Twain based on sys module |

> All files have been tested and compiled on Ubuntu 14.04 LTS
