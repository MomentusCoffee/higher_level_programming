# Python - Exceptions

### Description
In this project lesson you will learn:

* What’s the difference between errors and exceptions

* What are exceptions and how to use them

* When do we need to use exceptions

* How to correctly handle an exception

* What’s the purpose of catching exceptions

* How to raise a builtin exception

* When do we need to implement a clean-up action after an exception

### File Content
This repository contains the following files*:

| File | Task |
| :--- | :--- |
| 0-safe_print_list.py | A function that prints x elements of a list from 0-main.py |
| 1-safe_print_integer.py | A function that prints an integer with {:d}.format() from 1-main.py |
| 2-safe_print_list_integers.py | A function that prints the first x elements of a list and only integers from 2-main.py |
| 3-safe_print_division.py | A function that divides 2 integers and prints the result from 3-main.py |
| 4-list_division.py | A function that divides element by element 2 lists from 4-main.py |
| 5-raise_exception.py | A function that raises a type exception from 5-main.py |
| 6-raise_exception_msg.py | A function that raises a name exception with a message from 6-main.py |
| 100-safe_print_integer_err.py | A function that prints an integer from 100-main.py |
| 101-safe_function.py | A function that executes a function safely from 101-main.py |

> All files have been tested and compiled on Ubuntu 14.04 LTS
